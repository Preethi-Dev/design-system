# With Icons

You can use your own custom icons by setting the property `icon` for `items`.

<Canvas of={StepsStories.WithIcon} />

```jsx
const items = [
  {
    title: "Login",
    status: "finished",
    icon: "UserOutlined",
  },
  {
    title: "Verification",
    status: "finished",
    icon: "SolutionOutlined",
  },
  {
    title: "Pay",
    status: "current",
    icon: "LoadingOutlined",
  },
  {
    title: "Done",
    status: "wait",
    icon: "CheckOutlined",
  },
];
```
